Architecture Breakdown

Infrastructure Provisioning

Terraform automates EC2 instance, security groups, and key pair creation on AWS.

Configuration & Deployment

User data scripts and SSH provisioning automate Docker and application setup on the server.

Containerized Application

Docker Compose runs Redmine, MySQL, and supporting services in isolated containers.

Access & Networking

Security groups and port mapping expose the application securely via EC2 public IP.

Architecture Overview

The architecture demonstrates a fully automated deployment pipeline from infrastructure provisioning to running application containers in a secure AWS environment.
