Architecture Breakdown

☁️ AWS Infrastructure

AWS EC2 hosts the monitoring and automation platform. Terraform is used to provision and manage the infrastructure.

🐳 Container Layer

Docker Compose deploys and manages Prometheus, Grafana, Uptime Kuma, and n8n services.

📊 Monitoring Layer

Uptime Kuma monitors service availability, Prometheus collects metrics, and Grafana visualizes dashboards.

⚙️ Automation Layer

n8n receives monitoring events and automates the incident response workflow.

🤖 AI Layer

Ollama processes incident data and generates AI-powered summaries with recommended actions.

📱 Notification Layer

Telegram receives real-time incident alerts and service recovery notifications.
